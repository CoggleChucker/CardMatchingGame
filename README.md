## Installation

The project is created using Unity 2022 LTS with Universal 3D URP pipeline. Following are the steps to import the project

Step1: Download files and extract the zip

Step2: Inside Unity hub, select "add->add project from disc" option

Step3: select the folder in which the files are extracted

Step4: open newly created project



## Documentation

1. Change grid size
   
      To change the grid size, select "UI->GameplayMenu->CardsHolder" object

      Change the number of rows from "SetupGrid" script
  
      To change number of colums, open the script and change the private variable called "numberOfColums"
  
      <b><i>Note: Keep in mind that the multiplication of colums and rows must be even.</i></b>
