## Installation

The project is created using Unity 2022 LTS with Universal 3D URP pipeline. Following are the steps to import the project

Step1: Download files and extract the zip

Step2: Inside Unity hub, select "add->add project from disc" option

Step3: select the folder in which the files are extracted

Step4: open newly created project
